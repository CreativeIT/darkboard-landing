Design features

Material Dashboard is the first admin theme with the dark material design concept. It was a challenging project for us to design dashboard in dark material style, make data readable and usable. So here it is and we are proud to present you this powerful tool. This pack contains a nice portion of:

  • 8 useful widgets, 2 dashboard examples
  • Charts, Forms
  • Maps and Google maps markers with dynamic info
  • Components: Toggles, Sliders, Tooltips, Lists, Progress bars, Chips
  • UI elements: Buttons, Cards, Colors, Icons, Tables, Typography, Grid
  • Basic tables and customized Ag-grid
  • Pages: Sign In, Sign Up, Forgot password, 404
  • Filter and pagination grid, custom scroll, right sidebar
